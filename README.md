## Yo (via text)

Yo at a buddy with `yo <user>`

Signup users at `/create`.

Host your own texting yo by cloning this repo and creating an info.py with the following values and creating a new heroku app.

```python
db_name = ""
db_user = ""
db_pass = ""
db_host = ""

twilio_number = ""
twilio_sid = ""
twilio_secret = ""
```
